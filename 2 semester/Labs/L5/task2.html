Разыграть шесть опытов по схеме Бернулли: опыт состоит из четырёх независимых испытаний, в каждом из которых вероятность появления события А равна 0,45.

Случайные числа выбрать из таблицы приложения 1 следующим образом: N+3i; (N-10)+2i (где N - номер варианта, i - номер опыта).

Опыты:

A1 = A A A A => P(A A A A) = P(A)^4 = 0.45^4 = 0.0410

A2 = !A !A !A !A => P(!A !A !A !A) = P(!A)^4 = 0.55^4 = 0.0915

A3 = A !A !A !A => P(A !A !A !A) = P(A) * P(!A)^3 = 0.45 * 0.55^3 = 0.0749

A4 = A A !A !A => P(A A !A !A) = P(A)^2 * P(!A)^2 = 0.45^2 * 0.55^2 = 0.0613

A5 = A A A !A => P(A A A !A) = P(A)^3 * P(!A) = 0.45^3 * 0.55 = 0.0501

A6 = !A A A A => P(!A A A A) = P(!A) * P(A)^3 = 0.55 * 0.45^3 = 0.0501

Здесь не все возможные комбинации, но опытов по условию только шесть, поэтому и комбинаций столько же. А так как вероятность одинаковая, то они уже с 6 опыта повторяются (5 и 6 идентичны)

p1 = 0.0410; p2 = 0.0915; p3 = 0.0749; p4 = 0.0613; p5 = 0.0501; p6 = 0.0501
